# Turbot

A Ruby IRC bot.

## Description

Turbot is built using the Ruby IRC framework [cinch](https://github.com/cinchrb/cinch)

## Building

```bash
$ bundle install
$ ruby turbot.rb
```


## Deployment

```export MEETUP_API_KEY=API_KEY_HERE```

or

[Heroku Config Vars](https://devcenter.heroku.com/articles/config-vars)
